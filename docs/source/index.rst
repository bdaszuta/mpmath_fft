mpmath_fft
==========

Arbitrary-precision FFT for mpmath -- Cooley-Tukey, radix-2, and
Bluestein chirp-Z FFT using mpmath types stored in numpy object
arrays.  Pure Python, no JIT, no numba.  Mirrors the
`pynalgo.fft` API and plan structure.

Source code: https://github.com/bdaszuta/mpmath_fft
