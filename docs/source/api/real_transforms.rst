Real-Valued Transforms
=======================

``rfft``, ``irfft``, ``hfft``, ``ihfft`` -- real-input and Hermitian-symmetric
transforms following numpy.fft conventions.  Wrappers that call the complex
FFT from ``mpmath_fft`` via lazy import to avoid circular dependencies.
